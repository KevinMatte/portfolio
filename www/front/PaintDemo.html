The paint demo is work in progress and is actively updated as I work on it.

 

The demo source is available on my GitHub page: https://github.com/KevinMatte/portfolio

 

It is a framework for demonstrating:

- full stack development with: MySql, Apache2, WSGI, Python 3, Flask, ReactJS, Material-UI, HTML5, Kubernetes, and Docker

- Linux (Ubuntu) administration & networking

 

I just started this work on February 5, 2019, but with a GitHub setup on January 21, 2019.

 

As of the afternoon of February 6, 2019, it currently demonstrates the full stack, but with very little ReactJS, and isn’t yet using Kubernetes. It just shows a ‘Hello World’ button.

 

In the future, I’ll also be adding AngularJS, EmberJS and hosting it in Amazon Web Services, all of which I’ve done before, but not for myself or a portfolio.
